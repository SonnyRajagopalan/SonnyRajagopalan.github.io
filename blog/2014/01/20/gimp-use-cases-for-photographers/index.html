GIMP Use Cases for Photographers

Jan 20th, 2014

I found GIMP 2.8 useful for the following workflow tasks that typical photographers will want:

  • Contrast mask
  • Image sharpening (using <Image>-Filters-Enhance-Sharpen or <Image>-Filters-Enhance-Unsharp mask)
  • Smart sharpening (when you don’t want that silly look that comes from sharpening everything)
  • Color balancing
  • Correcting muddy pictures (auto levels)
  • Correcting blemishes (scratches and other artifacts)
